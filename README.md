# Personal Website - Nicolas Escobar Velasquez

A professional academic website for Nicolas Escobar Velasquez, Ph.D. student in Statistical Science at Indiana University Bloomington.

## Structure

- `index.html` - Main website page
- `styles.css` - Styling

## Features

- Responsive design for all devices
- Professional academic styling
- Sections: About, Research Interests, Software, Publications, Media, Contact
- Embedded YouTube presentation
- Links to GitHub repositories (riemtan, riemstats) and Google Scholar

## Local Development

Open `index.html` in a web browser to view the site locally.

## Deployment

This site can be deployed to GitHub Pages, Netlify, or any static hosting service.
